# Smetana (Сметана)

Desktop приложение для строительных смет с 2D-редактором планировок. Построено на Rust с использованием [egui/eframe](https://github.com/emilk/egui).

Язык интерфейса: русский. Целевая платформа: Windows (включая слабое оборудование).

## Возможности

- **Модель «от точки»** — сначала размещаются точки, затем на их основе строятся комнаты, стены и проёмы как полигоны
- **Переопределение рёбер** — ручной ввод расстояний и углов по результатам замеров
- **Расчёт площади комнат** — формула Гаусса (по координатам или по замерам при наличии переопределений)
- **Площадь стен** — общая и чистая (за вычетом проёмов), отображается на подписях рёбер
- **Проёмы дверей/окон** — настраиваемые размеры, направление открывания двери, индивидуальные цвета
- **Вырезы** — вычитание полигонов из площади комнаты
- **Разделение рёбер** — клик по ребру в инструменте «Точка» вставляет новую точку, автоматически обновляя все контуры
- **Умное удаление точек** — удаление точки исключает её из контуров, а не каскадно удаляет объекты
- **Отмена/повтор** — история на основе снимков (до 100 записей)
- **Автосохранение** — проекты сохраняются в `saves/projects/` в формате JSON
- **Навигация** — масштабирование колесом мыши, перемещение средней кнопкой или Space+перетаскивание, клавиши WASD
- **Привязка** — точки привязываются к существующим точкам, рёбрам и сетке; Shift отключает привязку

## Инструменты

| # | Инструмент | Описание |
|---|------------|----------|
| 1 | Выбор | Клик для выделения, перетаскивание точек/подписей, Delete для удаления |
| 2 | Точка | Размещение точки (привязка к точкам, рёбрам, сетке) |
| 3 | Ребро | Клик по двум точкам для создания ребра |
| 4 | Вырез | Клик по 3+ точкам для вырезания из содержащей комнаты |
| 5 | Комната | Клик по существующим точкам для построения контура комнаты |
| 6 | Дверь | Клик по существующим точкам для построения полигона двери |
| 7 | Окно | Клик по существующим точкам для построения полигона окна |
| 8 | Стена | Клик по существующим точкам для построения полигона стены |
| 9 | Подпись | Клик для размещения текстовой аннотации |

## Сборка и запуск

Требуется Rust edition 2024 (nightly или свежий stable).

```bash
cargo build              # Сборка (debug)
cargo run                # Запуск приложения
cargo test               # Запуск тестов
cargo clippy             # Линтер
cargo fmt                # Форматирование кода
```
