# Smetana (Сметана)

Desktop приложение для строительных смет с 2D-редактором планировок. Построено на Rust с использованием [egui/eframe](https://github.com/emilk/egui).

Язык интерфейса: русский. Целевая платформа: Windows.

## Сборка и запуск

Требуется Rust edition 2024 (nightly или свежий stable).

```bash
cargo build              # Сборка (debug)
cargo run                # Запуск приложения
cargo test               # Запуск тестов
cargo clippy             # Линтер
cargo fmt                # Форматирование кода
```
